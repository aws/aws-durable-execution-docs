# Testing

The testing framework supports writing unit and integration tests for durable functions
without deploying to AWS.

## In this section

- [Basic Tests](basic-tests.md) Common patterns for asserting on function results and
    operation execution
- [Complex Workflows](complex-workflows.md) Testing branching logic, retries, and
    multi-step sequences
- [Stores](stores.md) Configuring and inspecting the state store in tests
- [Testing Modes](testing-modes.md) Running tests locally in-memory or against a
    deployed Lambda function
